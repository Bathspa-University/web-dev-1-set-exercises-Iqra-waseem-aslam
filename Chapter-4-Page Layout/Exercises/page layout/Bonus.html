HTML Tags

Tag Description
<p> p represents as a paragraph.
<body> body display all the contents of html.
<img> img represents an image into the HTML document.
<a> represents as a hyperlink.
<table> represents as a table.
<form> represents an a HTML form for user input.
<input> represents a input control.
<div> represents as a division or a section in an HTML document.
<header> represents as a header for a document or a section.

Property Representation
color assortments the color of text.
font-size arrays the size of the font.
background-color bundles the background color of an element.
margin arrays the margin of an element.
padding assortments the padding of an element.
border assortments the border properties of an element.
display defines about the displaying behavior of an element.
position defines about the positioning work of an element.
